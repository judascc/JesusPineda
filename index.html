Información general

Esta plataforma es una prueba web para encontrar contenido básico para conceptos de expresión gráfica, es una prueba piloto la cual estaremos mejorando cada día.

Definición

Es una forma de comunicación que nos permite expresar de manera sencilla las ideas que serian muy dificiles de explicar con palabras. En Tecnologia, la expresión grafica es sumamente importante debido a que nos permite representar formas precisas de objetos tecnologicos que se piensan diseñar y construir.
Dentro de este proceso los dibujos desempeñan el mas importante metodo empleado, para poder plasmar las ideas de diseñadores y despues poderlas reproducir de forma fisica.

El objeto principal del dibujo t'ecnico (expresión grafica), es transmitir una información clara, puntual y precisa, encaminada a la interpretación de un objeto determinado.
